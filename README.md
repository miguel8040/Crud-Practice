# Crud-Practice


Repository creation: this repository is created to realize a CRUD simulation / practice of two tables : 
* CT_Movies
* CT_Directors

## Initialization 

* 1. Run SQL script to create database, tables, and insert the default data
* 2. Replace the conection string on the backend repository to could connect to your database
* 3. Run backend to deploy the API
  * In case the port is already used you should change the environment url on frontend
* 4. Install dependencies on frontend and run
